What It Is

An OS.
In Figma.

Crystal UI is a fully interactive phone OS prototype — not a theme, not a mockup. Every screen, transition, and interaction is built natively in Figma.

🔷

Glassmorphic Design System

Every element uses frosted glass layers, depth shadows, and translucency — creating a premium OS aesthetic throughout.

🕐

Depth Lock Screen

The clock renders behind and in front of the wallpaper simultaneously, giving a real 3D parallax depth effect.

⚡

Fully Interactive

Navigate between the lock screen, home screen, and apps — all working prototype interactions built in Figma.

📱

Real App Icons

Custom-designed icons for Spotify, Netflix, Discord, Google, Amazon, Chrome and 60+ more apps, all cohesive.

🎨

Fluid Wallpaper

The signature woven-fabric 3D wallpaper is consistent across both lock and home screens for a seamless visual identity.

🧩

Component-Based

Every screen is built from reusable Figma components — editable, scalable, and ready for your own customisation.
