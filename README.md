CrystalUI is a premium mobile OS design kit built entirely in Figma. It recreates the full experience of a modern smartphone operating system — including the lock screen, home screen, app icons, widgets, and system UI elements — all designed around a sleek glassmorphism aesthetic.
The kit features deep blue, fluid wallpapers paired with frosted glass panels, translucent app icons, and large bold typography for the clock and date display. The lock screen includes a minimal layout with a centered unlock ring, quick-access flashlight and camera buttons, and a clean date/time presentation. The home screen carries the same glass-forward design language, with rounded icon grids, a frosted search bar, and a customizable dock at the bottom.
Key features include:

Full lock screen and home screen templates
Custom-styled app icon set with rounded glass aesthetics
Glassmorphism UI components (cards, dock, search bar, status bar)
Deep blue ambient wallpapers designed to complement the UI
Large, modern clock widgets with bold and thin weight variants
Modular layout — easily swap icons, wallpapers, and widgets
Built with Figma Auto Layout and components for easy customization
Perfect for concept design, portfolio mockups, and UI presentation

CrystalUI is ideal for UI/UX designers, app concept creators, and anyone looking to present a polished, cohesive phone aesthetic without building from scratch.
